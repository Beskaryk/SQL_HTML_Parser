# Streamlit Data Utilities

Инструментарий для локальной обработки и анализа данных, включающий модули для работы с реляционными наборами данных и парсинга структурированных HTML-документов.

## Описание проекта

Проект состоит из двух независимых модулей, объединенных общим веб-интерфейсом:
* **SQL Парсер**: Позволяет загружать табличные данные (CSV) и выполнять аналитические запросы к ним, используя стандартный синтаксис SQL. Реализована функция выгрузки результатов обработки.
* **Парсер закладок браузера**: Инструмент для извлечения метаданных из HTML-дампов браузерных закладок. Извлекает структуру папок, URL-адреса, даты добавления и преобразует их в плоский табличный формат для последующей фильтрации и анализа.

## Стек технологий

* Python 3
* Streamlit
* Pandas / Pandasql
* BeautifulSoup4

## Структура проекта

- `main.py` — исходный код приложения.
- `requirements.txt` — список зависимостей проекта.
- `.gitignore` — исключения для системы контроля версий.
- `data/` — директория для тестовых файлов (содержит `sample_database.csv` и `sample_bookmarks.html`).

## Установка и запуск

1. Клонируйте репозиторий и перейдите в его директорию:
   ```bash
   git clone <URL_репозитория>
   cd <имя_папки>
   ```

2. Создайте и активируйте виртуальное окружение:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустите приложение:
   ```bash
   streamlit run main.py
   
```

## Использование

Для проверки функционала загрузите демонстрационные файлы из папки `data/` через веб-интерфейс. В модуле SQL Парсера обращение к загруженным данным осуществляется через таблицу `df` (например, `SELECT * FROM df LIMIT 10`).
